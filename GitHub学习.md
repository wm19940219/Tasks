# 进入

和 **Markdown** 相比，**GitHub** 可就难“入门”了，这个“入门”怎么说呢，真的是入门  
1.首先，有些人连官方网站都不一定找到！  
- 我是直接在浏览器搜索 **GitHub** 找到的（官方网站会有黑白🐱头像）
  
2.其次就是注册登录。新朋友需要注册，而填完基本信息后会有一个验证页面
  - 视觉验证：两张图片，图中有红绿两条绳子，周围有图标  💡有些图标会加粗
    >👉验证方式：左边绳子终点对应图标，右边在五张图里选正确连接相应图标的图片（我现在回想真是不甘心我被加粗图标误导，卡在这一步这么久😫）
  - 声音验证？（我没有选择这个方式，具体不是很清楚，新朋友有选这个的，很抱歉，我帮不到你）
    
3.登录成功以后首先就是建立自己的账户
